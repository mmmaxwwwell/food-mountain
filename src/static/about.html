About
  • I created this website as a way to keep track of recipes I enjoy. I always lose or somehow destroy paper copies, so I figured a website would be safer.
  • You will never encounter ads or tracking cookies on this website. Ever.
  • This entire site, including the repo is MIT licensed. You are free to steal, but please contribute.
  • I'm hosting this website through a service called netlify. I'm on the free tier, so lets hope this doesn't get too popular.
  • If you want to submit a recipe, go on over to the github repo and open an issue or a pull request.
